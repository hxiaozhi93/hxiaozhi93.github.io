YOLO

5 May 2017

实践出真知。YOLO是基于回归的快速目标检测方法。在此，尝试在windows+CPU+网络摄像头的基础上调试YOLO基于实时的目标检测。

  • 目标检测
  • YOLO
  • Windows+CPU
  • Python
  • 网络摄像头

论文《You Only Look Once_ Unified, Real-Time Object Detection》

主页：https://pjreddie.com/darknet/yolov1/

代码：https://github.com/jerrygood0703/caffe-yolo

下载代码，下载模型如yolo_tiny.caffemodel

输入命令：python yolo_main.py -m yolo_tiny_deploy.prototxt -w yolo_tiny.caffemodel -i cat.jpg

结果：通过摄像头进行实时检测，以边界框和得分概率形式给出检测结果。

按Esc键或q键结束检测。

小白。勿念。
