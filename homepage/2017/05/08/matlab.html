MATLAB相关基础指令

应用于通信系统相关函数：

poly2trellis：Convert convolutional code polynomial to trellis description.

gf：Create a Galois field array.

rsenc： Reed-Solomon encoder.

rsdec： Reed-Solomon decoder.

convenc： Convolutionally encode binary data.

vitdec: Convolutionally decode binary data using the Viterbi algorithm.

pskmod: Phase shift keying modulation

qammod: Quadrature amplitude modulation

pskdemod: Phase shift keying demodulation

qamdemod Quadrature amplitude demodulation.

complex: Create complex array.

ifft: Inverse discrete Fourier transform.

fft: Discrete Fourier transform.

awgn: Add white Gaussian noise to a signal.

semilogy: Semi-log scale plot.

randint: Generate matrix of uniformly distributed random integers.

biterr: Compute number of bit errors and bit error rate.

help/在线帮助,demo /运行演示程序,ver/版本信息,who/显示当前变量,whos/显示当前变量的详细信息,clear/清空工作间的变量和函数,load/把文件调入变量到工作间,save/把变量存入文件中,quit/exit/退出MATLAB ,cd/改变当前的工作目录,pwd/显示当前的工作目录,dir /显示目录内容,computer/显示计算机类型,

在执行关系及逻辑运算时，MATLAB 将输入的不为零的数值都视为真 (True)而为零的数值则视为假 (False)。 运算的输出值将判断为真者以 1 表示而判断为假者以 0 表示。

通过在矩阵变量后加’的方法来表示转置运算

M 文件，以.m 为扩展名，所以称为M 文件。M文件是由一系列MATLAB语句组成的文件，包括命令文件和函数文件两类，命令文件类似于其他高级语言中的主程序或主函数，而函数文件则类似于子程序或被调函数。数据文件，以.mat 为扩展名，所以又称MAT文件。在讨论工作空间窗口时已经涉及到MAT文件。显然，数据文件保存了MATLAB工作空间窗口中变量的数据。图形文件，以.fig 为扩展名。主要由MATLAB 的绘图命令产生，当然也可用File菜单中的New命令建立。MEX 文件，以.mex 或.dll 为扩展名，所以称MEX 文件。MEX 实际是由MATLAB Executable 缩写而成的，由此可见，MEX 文件是MATLAB 的可执行文件。模型和仿真文件，模型文件以.mdl为扩展名，由Simulink仿真工具箱在建立各种仿真模型时产生。仿真文件以.s 为扩展名。
